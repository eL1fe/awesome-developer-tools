# Contributing

Thanks for your interest in contributing! Here's how you can help:

## Adding a new tool

1. Fork this repository
2. Add your tool to the appropriate category in `README.md`
3. Follow the existing format:
   ```markdown
   | [Tool Name](https://url.com) | Brief description (one sentence) | Pricing |
   ```
4. Ensure the tool is:
   - Actively maintained
   - Useful for developers
   - Not a duplicate of an existing entry
5. Submit a pull request

## Categories

If you think a new category should be added, open an issue first to discuss it.

## Guidelines

- Keep descriptions concise (1-2 sentences max)
- Use `Freemium`, `Free`, or `Paid` for pricing
- Tools should be generally available (no private betas)
- No affiliate links

## Questions?

Open an issue and we'll get back to you!
